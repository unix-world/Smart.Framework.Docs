Class SmartCryptoMCrypt

Class: SmartCryptoMCrypt Provides a MCrypt based feature to handle the Blowfish (CBC / ECB) encryption / decryption.

This provides an advanced crypto handler for Blowfish CBC / ECB algorithm(s). WARNING: The ECB encryption is weak, have no Initialization Vector. You should not use it if you don't know what you really do ! It is recommended to use instead the SmartUtils::crypto_blowfish_encrypt() and SmartUtils::crypto_blowfish_decrypt() which are detecting if to use the MCrypt or built-in Blowfish classes and use always a compatible layer of Blowfish CBC.

Final

Hints: Blowfish is a 64-bit (8 bytes) block cipher. Max Key is up to 56 chars length (56 bytes = 448 bits). The CBC mode requires a initialization vector (iv).
