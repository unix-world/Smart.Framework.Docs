Class SmartAbstractPersistentCache

Class: Smart.Framework Abstract Persistent Cache. The backends used for Persistent Cache must be very fast, must support large keys and must supply key expiration by time. If the key expiration is not supported natively, then a custom function must be created to delete expired keys.

It must contain ONLY public functions to avoid late state binding (self:: vs static::)
