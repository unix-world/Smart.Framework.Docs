Parameters

$y_code
The code for the BarCode Generator
$y_type
The BarCode Type: qrcode / semacode / pdf417
$y_format
The Barcode format: html, html-png, png, html-svg, svg
$y_size
The Scale-Size for Barcode (1..4)
$y_color
The Hexadecimal Color for the Barcode Pixels ; default is Black = #000000
$y_extraoptions
Extra Options: for QRCode = Quality [L, M, Q, H] L as default ; for PDF417 a Ratio Integer between 1 and 17
$y_cache
If YES will cache the Barcode to avoid on-the-fly generation ; default is set to NO
